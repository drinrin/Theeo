# Theeo
Discord Music Bot

# Overview
Theeo plays music for a discord server. This is a personal project and Theeo is used in my private discord server. <br>
It was programmed using JavaScript and deployed using Heroku.

# Commands
<ul>
  <li>/play</li>
  <li>/queue</li>
  <li>/skip</li>
  <li>/skipto</li>
  <li>/shuffle</li>
  <li>/info</li>
  <li>/quit</li>
</ul>
